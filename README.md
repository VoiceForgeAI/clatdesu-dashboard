CLATdesu Dashboard - Vercel-ready (Final)

Features:
- Gradient blue theme (auto dark/light)
- Password: clatdesu (prompt-based session login)
- Daily tracker with timer (start/pause/stop & save minutes)
- Syllabus progress, checklist, mock scores, streak counter
- Local backup (export JSON) and Export to Print/PDF

To deploy on Vercel:
1. Push this folder to your GitHub repository (yourrepo/clatdesu-dashboard)
   - From GitHub website: Upload files -> commit directly to main branch
2. In Vercel: Import Project -> select your GitHub repo -> Deploy
   - Framework: Vite, Build Command: npm run build, Output: dist
3. Open the site and enter the password: clatdesu

Or: use Vercel manual upload page and upload the ZIP directly.
